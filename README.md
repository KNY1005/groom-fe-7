**2025년 11월 29일**부터 과정을 시작하여, HTML과 CSS의 기초 문법부터 다양한 실습 예제까지 단계별로 기록하고 있습니다.

---

##  레포지토리 소개

이 저장소는 수업 시간에 실습하며 작성한 예제 코드와 복습용 노트를 정리한 리마인드용 레포지토리입니다.  
각 폴더에는 실제로 실습한 코드가 포함되어 있으며,  
주석과 마크다운 문서를 통해 개념을 되짚어볼 수 있도록 구성되어 있습니다.

---

## 주요 학습 내용

- **HTML 기본 구조 및 시맨틱 태그 이해**
- **CSS 선택자와 속성 정리**
- **display / position / float 등의 레이아웃 속성 실습**
- **박스 모델(box-sizing) 및 Reset CSS 적용**
- **clearfix 개념과 float 문제 해결 방법**
- **background, font, text 관련 스타일링**
- **단위(px, em, rem, %) 및 뷰포트 단위(vw, vh) 이해**

---

## 학습 목적

- CSS의 기초 개념을 체계적으로 익히기 위해 정리한다.  
- 코드 중심 실습을 통해 스타일 속성이 실제로 어떻게 작동하는지 확인한다.  
- 수업 중 배운 내용을 개인적으로 복습하고, 나중에 참고할 수 있는 노트를 만든다.  
- 프론트엔드 개발의 기초를 탄탄히 다지기 위한 기록물로 남긴다.
